https://api.eldis24.ru/ Общая информация по API

Для работы с API необходимо в каждом запросе передавать ключ доступа к API.Ключ передается в заголовке запроса с именем key. Все даты, передаваемые/получаемые через json формируются как TimeStamp.

Глобальные переменные передаваемые через URL:
Параметр Описание Тип данных
suppressResponseCode Подавляет код ответа http. Если указать значение параметра true, то http ответ всегда будет возвращаться со статусом 200 – ОК. Значение по умолчанию – true. bool
limit Максимальное количество возвращаемых записей для одного запроса. Значение по умолчанию – 50. Максимальное количество возвращаемых записей – 500. int
page Номер страницы, по которой будет возвращено указанное количество записей параметром limit. Значение по умолчанию – 1. int
search Осуществляет поиск записей. string
entity Название сущности/метода для которой применяется вложенный запрос. string

https://api.eldis24.ru/api/v1/users/login Производит авторизацию пользователя в системе

Все методы API будут доступны лишь при удачной авторизации пользователя. При удачной авторизации пользователя в cookie файле будет возвращен токен доступа access_token, который необходимо передавать для вызова других методов API. Время жизни токена 1 час, после чего токен будет обновлен и возвращен повторно при следующей активности пользователя.

Headers:


    
Content-Type:
 application/x-www-form-urlencoded
    
key
 = "you_key_here"
    
Cookie:
 access_token= "you_token_here"

Parameters:
Параметр Передавать через Описание Тип данных
login form Логин пользователя string
password form Пароль пользователя string

Response:


    {
      "response": {
        "users": {
          "login": {
            "result": "bool" // Результат авторизации
          }
        },
        "messages":  []  // Список сообщений
      }
    }

limit url Количество записей, выводимых на одной странице. Максимальное значение – 500. Значение по умолчанию – 50. int
page url Номер отображаемой страницы. Значение по умолчанию – 1. int
search url Осуществляет поиск данных string

Form's parameters:
Параметр Передавать через Описание Тип данных
notUseIDs form Массив ID тегов, которые стоит исключить из поиска данных. ID тегов передавать через разделитель | Guid array
useIDs form Массив ID тегов, которые стоит найти для получения названия. ID тегов передавать через разделитель | Guid array
recordIDs form Массив ID записей для поиска повторяющихся тегов среди переданного массива. ID записей передавать через разделитель | Guid array
